+ account creation
+ setting of tasks
+ setting tasks due at certain dates/times
+ ability to set alerts
+ ability to set extra information for a task
+ minimalistic UI/UX
+ autopropagation to realtime database
+ notification/alert intents that open to app
+ management of tasks
